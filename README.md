# Science Journal UI (React + TS + Vite)

Базовая архитектура для научного журнала с тремя ролями: авторы публикуют статьи, редакторы ведут очередь, рецензенты работают в слепом режиме.

## Как запустить

- `npm install` — установка зависимостей (уже выполнено).
- `npm run dev` — локальная разработка.
- `npm run build` — проверка типов и сборка.

## Что внутри

- React Router настроен на четыре ключевых маршрута: обзор, статьи автора, очередь редакторов, назначения рецензентов.
- Файловая структура по ролям и сущностям: `src/features/{authors,editors,reviewers,articles}`, общие типы в `src/shared/types.ts`, мок-данные в `src/data/mockData.ts`.
- Компонент `MainLayout` задаёт шапку, навигацию и shell; `StatCard`, `Badge`, `ArticleCard` переиспользуются в разных разделах.
- Макет и стили ориентированы на журнал: акцентные карточки, статусные бейджи, упор на контроль слепых рецензий.

## Возможные следующие шаги

- Подключить реальный API/сервер данных, заменить мок-структуру.
- Добавить формы подачи статьи и назначения рецензентов.
- Ввести аутентификацию и разграничение доступа по ролям.
